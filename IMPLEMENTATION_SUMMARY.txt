🧮 REPUTATION FORMULA
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Score = (successful_repays / total_loans) × 100

  Status Classification:
  ┌──────────────────────────────────────────┐
  │ New Agent      → total_loans == 0        │
  │ Verified       → score >= 70 (trusted)   │
  │ Established    → 50 <= score < 70        │
  │ Risky          → score < 50 (risky)      │
  └──────────────────────────────────────────┘

  Activity Level:
  ┌──────────────────────────────────────────┐
  │ Low    → 1-4 transactions               │
  │ Medium → 5-9 transactions               │
  │ High   → 10+ transactions               │
  └──────────────────────────────────────────┘

  POST /loan/wallet/request
      ↓

      ↓

  5. Treasury: Check fund availability
      ↓
  6. Update agent metrics ← KYA EVENT HANDLER

      ├─ Update activity level
      └─ Update trust score
      ↓
  7. Return loan decision
